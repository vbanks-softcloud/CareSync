# Scripts

Repo-wide helper scripts (Node, bash, PowerShell).

Examples of things that belong here:

- `bootstrap.sh` — one-time setup for new contributors
- `release.sh` — tag + changelog generation
- `clean.sh` — wipe all `node_modules`, `dist`, `.serverless`, etc.

Service-specific scripts should live with the service (e.g. `backend/scripts/`).
